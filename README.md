# vue2-玩具商城-前端静态网站
### 1、介绍
玩具商城前端静态网站是一个基于 Vue2 的电商前端项目，采用纯前端静态数据模拟方式实现完整的电商购物流程。项目涵盖用户注册登录、商品浏览与搜索、购物车管理、订单结算、个人中心、社区互动、帮助中心等核心功能模块，共15个页面视图、2个公共组件、15条路由！
